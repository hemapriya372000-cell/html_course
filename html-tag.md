📑 Basic Structure Tags
<html> – Root element of an HTML document

<head> – Metadata container (title, scripts, styles)

<title> – Page title (shown in browser tab)

<body> – Main content of the page

📝 Text & Formatting
<h1> to <h6> – Headings (largest to smallest)

<p> – Paragraph

<br> – Line break

<hr> – Horizontal rule (divider)

<b> / <strong> – Bold text

<i> / <em> – Italic/emphasized text

<u> – Underlined text

<small> – Smaller text

<mark> – Highlighted text

<sup> / <sub> – Superscript / subscript

🔗 Links & Media
<a> – Hyperlink

<img> – Image

<audio> – Audio file

<video> – Video file

<source> – Media source for <audio> or <video>

<iframe> – Embed another webpage

📊 Lists & Tables
<ul> – Unordered list

<ol> – Ordered list

<li> – List item

<table> – Table

<tr> – Table row

<td> – Table cell

<th> – Table header cell

<caption> – Table caption

🧩 Forms & Input
<form> – Form container

<input> – Input field (text, password, checkbox, radio, etc.)

<textarea> – Multi-line text input

<button> – Clickable button

<select> – Dropdown menu

<option> – Option in dropdown

<label> – Label for form elements

<fieldset> – Group related form controls

<legend> – Caption for <fieldset>

📦 Semantic & Structural
<header> – Page or section header

<footer> – Page or section footer

<nav> – Navigation links

<section> – Thematic grouping of content

<article> – Independent content (blog post, news item)

<aside> – Sidebar or tangential content

<main> – Main content area

<div> – Generic container

<span> – Inline container

⚙️ Metadata & Scripting
<meta> – Metadata (charset, viewport, etc.)

<link> – Link to external resources (CSS, icons)

<style> – Internal CSS styles

<script> – JavaScript code
